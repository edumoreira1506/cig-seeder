# cig-seeder

## Dependencies

```
NodeJS >= 16.14
```

## Setup

Clone the project.

```console
$ git clone git@github.com:edumoreira1506/cig-seeder.git
$ cd cig-seeder
```

Set the urls on the env file.

```console
$ cp .env.sample .env
```

Install all packages.

```console
$ npm i
```

For run seed command.

```console
$ npm run seed
```
